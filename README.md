## How to run AwaQuiz App

- Clone the repo and `cd` into the folder
- Run `composer install`
- Run `npm install`
- Run `npm run build`
- Run `./vendor/bin/sail up -d`, Docker is required
- Run `./vendor/bin/sail artisan migrate --seed`, Docker is required
